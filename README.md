# 我的第一个原生小程序可开发 
### 首页显示当前教室的情况
1. 首页显示当前教室的人数、温度、湿度、亮度
2. 硬件上采用红外模块高低电平统计经过的人数，温湿度分别通过具体的温度传感器、湿度传感器传输到硬件上，亮度采用光敏传感器进行模数转换获取到数值
### 通知功能
1. 点击搜索框左侧的发布按钮就可以进入到发布的具体界面，发布界面实现微信统计最多字数不能超过140个字，图片最多只能上传九张
### 个人信息
1. 个人信息功能采用动态的波纹图，呈现蓝色边框底部动态流动的效果
